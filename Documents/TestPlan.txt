Тестирование API. Проверить операции через http-запросы (web api):

Проверка функционала:
+Получение информации о пользователе по заголовку Authorization.
+Поиск агентов.
+Добавление агента.
+Изменение агента.
+Удаление агента.
+Поиск фирмы.
+Добавление фирмы.
+Изменение фирмы.
+Удаление фирмы.
+Поиск объектов недвижимости.
+Добавление объекта недвижимости.
+Изменение объекта недвижимости.
+Удаление объекта недвижимости.
+Поиск фотографий.
+Добавление фотографии.
+Изменение фотографии.
+Удаление фотографии.
+Проверить, при каждом ли действии происходит проверка пользователя.

Проверка авторизации (без учета прав глобального администратора):
+Проверить, что агент может добавлять, удалять и изменять только свои квартиры и 
	квартиры своих подчиненных.
+Проверить, что добавлять и удалять агентов фирмы может только админ фирмы.
+Проверить, что изменять данные агента может только сам агент или админ фирмы.
+Проверить, что фотографии объекта недвижимости может редактировать только владелец 
	объекта недвижимости.
+Проверить, что фотографии фирмы может редактировать только админ фирмы.



Найденные (неисправленные) ошибки и вопросы:
-Вообще не реализована валидация.
-Можно добавить двух агентов с одинаковыми именами и телефонами.
-Можно добавить двух агентов - администраторов для одной и той же фирмы.
-Ошибка "Ссылка на объект не указывает на экземпляр объекта" при попытке удаления
	несуществующего агента (нет валидации).
-У фирмы нет никаких данных кроме названия.
-Если при запросах PUT (для разных контроллеров) не указать id в самом JSON'е, 
	данные не обновляются корректно (даже если id присутствует в URL).
-Ошибка при обновлении фирмы (PUT) (при вызове this.Context.Firms.Attach() в FirmsRepository).
-Отсутствует возможность получить фотографии объекта недвижимости или конкретной фирмы.
-В БД у фотографий нет поддержки null-значений для RealtyObjectId и FirmId, фотографии не
	добавляются, не удаляются и не изменяются, т.е. API фотографий не работает.


Исправленные ошибки:
+Можно добавить двух агентов с одинаковыми именами и телефонами.
+У фирмы нет никаких данных кроме названия.
+Ошибка при обновлении фирмы (PUT) (при вызове this.Context.Firms.Attach() в FirmsRepository).
+Отсутствует возможность получить фотографии объекта недвижимости или конкретной фирмы.
+В БД у фотографий нет поддержки null-значений для RealtyObjectId и FirmId, фотографии не
	добавляются, не удаляются и не изменяются, т.е. API фотографий не работает.
